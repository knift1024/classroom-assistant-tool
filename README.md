## 專案簡介

這是一款專為**教學、簡報和線上會議**設計的輕量級桌面輔助工具。它將三個最常用的功能——**螢幕畫記**、**計時器**和**隨機抽籤**——整合到一個浮動的、可收合的迷你工具列中，讓您在不干擾主要畫面的情況下，隨時取用強大的輔助功能。

本工具以 **輕量、整合、不干擾** 為核心設計理念，所有設定（如視窗位置、畫筆顏色、抽籤名單）都會自動儲存，提供極佳的使用體驗。

## 主要功能

### 1. 核心工具列
- **浮動置頂**：永遠顯示在所有視窗的最上層，方便隨時點擊。
- **可拖曳與位置記憶**：自由拖曳到螢幕任何角落，程式會記住您上次關閉時的位置。
- **動畫收合介面**：點擊箭頭可流暢地展開或收合功能按鈕，最大化節省螢幕空間。

---

### 2. 螢幕畫記
啟動後，主工具列會自動隱藏，讓您專注於畫記。

 <!-- 建議替換成您自己的截圖 -->

- **多種繪圖工具**：包含手繪、直線、箭頭、矩形、圓形。
- **雷射筆模式**：具備動態寬度和淡出殘影效果，模擬真實雷射筆的軌跡。
- **多樣畫布背景**：可在當前桌面、黑板、白板或自訂純色背景上畫記。
- **輔助樣式**：可開啟細/粗方格，方便對齊和繪圖。
- **完整功能**：支援復原/重做、調整顏色與粗細、筆觸平滑化、儲存畫記 (可存全螢幕或圈選範圍)。
- **獨立工具列**：畫記工具列本身也可鎖定位置，避免誤觸。

---

### 3. 計時器

 <!-- 建議替換成您自己的截圖 -->

- **倒數計時**：設定分鐘和秒數進行倒數，時間到會發出提示音並彈出提醒視窗。
- **碼錶**：支援開始、暫停、繼續，並可隨時「計次」(Lap) 記錄時間點。

---

### 4. 隨機抽籤

 <!-- 建議替換成您自己的截圖 -->

- **姓名抽籤**：可貼上名單，設定抽出人數，並可選擇抽出後是否放回（不重複抽籤）。
- **數字抽籤**：在指定數字範圍內進行抽籤，支援分組、設定每組人數、以及是否可重複抽。

## 近期更新與效能優化 (2025-09-07)

本次更新主要針對應用程式的渲染效能進行了大幅優化，解決了在特定操作下可能發生的延遲與卡頓問題。

### 1. 手繪工具效能優化

-   **問題分析：** 當畫布上的筆劃增多時，手繪工具的效能會明顯下降。經過分析，根本原因在於每一次滑鼠移動都導致了整個畫布的完整重繪。
-   **解決方案：** 現已優化繪圖邏輯，僅針對正在變動的局部區域進行重繪。這項改動大幅減少了不必要的渲染運算，即使在複雜的畫布上也能確保流暢的繪圖體驗。

### 2. 工具列動畫效能優化

-   **問題分析：** 部分工具列按鈕的翻轉動畫可能會在低效能電腦上觸發延遲。
-   **解決方案：**
    -   將工具列的渲染處理設定為獨立的原生視窗，使其動畫不再觸發主畫布的重繪。
    -   將動畫時間減半，提供更靈敏的視覺回饋。

## 如何使用

### (A) 對於一般使用者 (執行 .exe)
1.  下載後直接雙擊執行即可，無需安裝。

### (B) 對於開發者 (從原始碼執行)

**環境需求:**
- Python 3.7+

**步驟:**
1.  複製本專案：
    ```bash
    git clone https://github.com/your-username/your-repo.git
    cd your-repo
    ```
2.  建立並啟用虛擬環境 (建議)：
    ```bash
    # Windows
    python -m venv .venv
    .\.venv\Scripts\activate
    ```
3.  安裝相依套件：
    ```bash
    pip install -r requirements.txt
    ```
4.  執行主程式：
    ```bash
    python main.py
    ```

## 如何打包成執行檔

本專案已對 PyInstaller 進行優化，可輕鬆打包。
1.  確保已安裝 PyInstaller：
    ```bash
    pip install pyinstaller
    ```
2.  在專案根目錄下執行以下指令：
    ```bash
    pyinstaller --noconsole --onefile --windowed --name "ClassroomTools" main.py
    ```
    - `--noconsole` 或 `--windowed`：建立無主控台視窗的 GUI 應用。
    - `--onefile`：將所有檔案打包成單一 `.exe` 檔。
    - `--name`：指定輸出的執行檔名稱。

    打包完成後，執行檔會位於 `dist` 資料夾內。
